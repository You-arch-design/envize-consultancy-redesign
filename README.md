# Envize Consultancy – UI/UX Redesign

A modern UI/UX redesign project for the Envize Consultancy website focused on improving responsiveness, user experience, visual hierarchy, and professional frontend design while preserving the original business content and services.

## 🚀 Project Overview

This project redesigns the existing Envize Consultancy website with a clean, modern, and responsive corporate interface. The goal was to improve the overall user experience, accessibility, responsiveness, and visual appearance of the website without changing the original business information.

## ✨ Features

* Modern and professional UI/UX design
* Fully responsive layout for desktop, tablet, and mobile devices
* Improved typography and spacing
* Modern hero section with strong CTA buttons
* Redesigned services section with interactive cards
* Sticky responsive navigation bar
* Smooth animations and hover effects
* WhatsApp floating contact button
* Improved footer and contact section
* Optimized user experience and readability
* Performance-focused frontend structure

## 🛠️ Technologies Used

* React
* Tailwind CSS
* HTML5
* CSS3
* JavaScript
* Framer Motion

## 🎯 Objectives

* Improve the visual design of the existing website
* Enhance responsiveness across devices
* Create a modern corporate web experience
* Improve usability and accessibility
* Maintain existing business identity and content
